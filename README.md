# 소현 국어 Final - React PWA

## 아이패드에서 쓰는 가장 쉬운 방법
1. 이 폴더를 GitHub 새 repository에 업로드
2. Vercel.com 가입 → Add New Project → GitHub repository 선택
3. Framework는 Vite로 자동 인식됨 → Deploy
4. 나온 링크를 아이패드 Safari로 열기
5. 공유 버튼 → 홈 화면에 추가

## Firebase 사용
`src/firebase.js`에 Firebase 설정값을 넣으면 Firestore 저장으로 바꿀 수 있습니다. 기본 버전은 localStorage 저장이라 로그인 없이 작동합니다.

## 로컬 실행
npm install
npm run dev
